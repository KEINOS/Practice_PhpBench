# Practice of PHPBench

Sample script to benchmark PHP functions using [PHPBench](https://github.com/phpbench/phpbench).

```shellsession
$ # Bench memory and time
$ composer bench-summary
...
$ # Bench memory
$ composer bench-memory
...
$ # Bench time
$ composer bench-time
...
```
